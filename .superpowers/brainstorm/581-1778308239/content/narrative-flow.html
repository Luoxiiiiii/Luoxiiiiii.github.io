设计·第三部分：叙事节奏与游戏流程

玩家怎么玩？故事怎么推进？

三幕结构（45-90分钟）

🎬 第一幕 · 发现（约15-25分钟）

目标：让玩家熟悉手机界面，建立神秘感
1. 打开手机，玩家先随便翻看各个app
2. 在消息里看到姐姐和一个未知号码的奇怪对话
3. 发现浏览器搜索历史和电台app的关联
4. 🌙 第一次切到深夜 → 电台里传来说话声
5. 第一幕结尾：玩家意识到"这个电台有问题"

🔍 第二幕 · 调查（约20-40分钟）

目标：深度挖掘，窥探器成为核心工具
1. 白天/深夜多次切换，发现不同时间段才有不同的内容
2. 开始使用窥探器搜索关键词，建立线索网
3. 发现更多联系人、更多网页、更多秘密
4. 拼凑出电台的真相——它是一个催眠诱导工具
5. 发现姐姐不是唯一的受害者

🌀 第三幕 · 反转（约10-20分钟）

目标：真相揭露 + 双重反转
1. 窥探器解锁最终线索 → 揭示电台背后组织的全貌
2. 剧情反转：主角发现自己也是被催眠者——"你为什么会拿到这部手机？真的是你主动想调查的吗？"
3. 元反转：手机屏幕开始闪烁、扭曲——催眠暗示出现在游戏界面本身
4. 尾声："现在你也是听众了。晚安。"

关键设计决策

  • 自由度 vs 线性：玩家可以随时打开任何app，但关键线索需要满足条件（特定的时间段、特定的关键词搜索）才会出现
  • 时间推进方式：手动切换，不需要等真实时间。但有些内容在切换后才会"解锁"
  • 游戏结束机制：没有"游戏失败"。所有路都通向同一个结局，但发现的内容多少会影响结局时的"冲击力"
  • 重玩性：部分隐藏线索第一次可能漏掉，但结局相同

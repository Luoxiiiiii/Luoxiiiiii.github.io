# 87.9 MHz

> 一款手机解谜游戏。你拿到了姐姐的手机，发现她最近行为异常。深入调查后，一个神秘深夜电台的真相逐渐浮现……

**游戏时长：** 约 20-40 分钟
**纯前端，无需服务器。**

## 如何游玩

1. 下载整个项目文件夹
2. 双击打开 `index.html` 即可开始
3. 支持 Chrome / Edge / Safari

或部署到 GitHub Pages / Netlify（直接拖拽项目文件夹即可）。

## 操作说明

- 点击 app 图标打开应用
- 点击 ← 返回主屏幕
- 点击顶部 **☀️白天 / 🌙深夜** 切换时间
- 使用 **窥探器** 输入关键词搜索线索

## 游戏流程

1. 翻阅手机中的消息、相册、备忘录等应用
2. 发现线索并解开密码锁
3. 切换至深夜模式收听神秘电台
4. 解锁窥探器，交叉搜索关键词
5. 拼凑出全部真相……

## 谜题提示

<details>
<summary>点击查看密码提示（有剧透）</summary>

- 相册密码：姐姐的生日（MMDD），在备忘录里找线索
- 备忘录密码：同上
- 窥探器关键词：试试 87.9、R-879、0520、救
</details>
